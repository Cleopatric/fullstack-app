### Backend setup

1. Run build:

    ```sh
   make build -B
    ```
  
2. After completed, stop the process and run command:

    ```sh
    make start
    ```

3. Create a superuser:

    ```sh
    make admin
    ```

4. Finally, open [Admin Panel](http://0.0.0.0:8000) and create some instances for cities

### Frontend setup
1. Install requirements:

    ```sh
     npm install
    ```
   
2. Run application:

    ```sh
     npm start
    ```
